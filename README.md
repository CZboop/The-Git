# The Git Happens Here 🖥️

Net art inspired ASCII terminal animation. Badly representing where the git happens 🤔

Made using the great [asciimatics package](https://github.com/peterbrittain/asciimatics).

## The Gif 📽️
Animation consists of two ASCII computers, connected by lines with arrows going back and forth, some figlet text and randomised sparkles.

Check it out in action:

![ascii terminal animation of two computers with arrows moving between them, and text that says "the git happens here"](./media/the_git.gif)

## Inspiration 💡
Inspired by [this piece of art](https://en.wikipedia.org/wiki/Internet_art#/media/File:Simple_Net_Art_Diagram.svg), that very helpfully illustrates exactly how Net Art works.
